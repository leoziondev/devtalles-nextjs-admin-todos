# NextJS 14 Admin Todos APP

## How to run this project

### Developmnet
1. Start database
```
docker compose up -d
```

2. Rename .env.example to .env
3. Replace environment variables
4. Execute SEED to [create local data db](localhost:3000/api/seed)


### Prisma commands
```
npx prisma init
npx prisma migrate dev
npx prisma generate
```
